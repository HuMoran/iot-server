# API文档

## 硬件终端登录

数据以十六进制BUFF发送
| 数据                 | 说明                                                |
| :------------------- | :-------------------------------------------------- |
| 06                   | 0：代表注册，6：代表ID长度                                              |
| 11 40 16 79 79 1f    | 终端ID号，奇位数末尾补'f'                           |
| 00                   | 群号长度，00代表没有群号                            |
|                      | 群号ID号，奇位数末尾补'f'，没有则省略               |
| ff                   | 登录方式：00: 备用IP或者域名登录，ff:主IP火域名登录 |
| 180413               | 出厂日期：年/月/日                                  |
| 4e 14 b8             | 校验码计算值                                        |
| 34372e37352e332e3638 | 服务器域名或IP                                      |
| ff                   | 分隔符                                              |
| 3630303030           | 端口号                                              |
| 1f                   | 校验码                                              |

示例数据：
> 0611401679791f00ff1804134e14b834372e37352e332e3638ff36303030301f
